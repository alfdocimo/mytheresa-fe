# Basic react starter 2020 ⚛️

The title says it all. One quick react starter using `babel` & `webpack`.

Think of this as the yeast of your project and get cooking 🍞🥖

Installing dependencies:

`yarn`

or

`npm i`

To run locally:

`yarn dev`

or

`npm run dev`

Building to prod:

`yarn build`

or

`npm run build`

Have fun!
